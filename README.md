🚖 Ola Ride Data Analysis

📌 Overview
This project analyzes Ola ride booking data to generate insights on booking trends, revenue, cancellations, and customer behavior. The analysis is performed using SQL, and results are visualized through an interactive Power BI dashboard.

🎯 Objectives
* Analyze total bookings and revenue trends
* Identify cancellation patterns (customer & driver)
* Evaluate ride demand over time
* Compare performance across vehicle types
* Understand customer and driver ratings

 🛠️ Tools Used
* SQL (MySQL)
* Power BI
* Microsoft Excel

📊 Dashboard
The Power BI dashboard includes:
* Total Bookings and Revenue KPIs
* Booking Status Breakdown
* Ride Volume Trends
* Interactive filters (Date, Vehicle Type)

🔍 Key Insights
* Majority of rides are successfully completed
* Cancellations occur from both customers and drivers
* Ride demand varies across different time periods
* UPI is a widely used payment method
* Ratings differ across vehicle categories

📁 Project Files
* `ola_analysis.sql` – SQL queries for analysis
* `ola_data_analysis_dataset.xlsx` – Dataset
* `ola_dashboard.pbix` – Power BI dashboard

🚀 How to Use
1. Import dataset into MySQL
2. Run queries from `ola_analysis.sql`
3. Open Power BI file to explore dashboard
